# TransitOps

Smart Transport Operations Platform

## Description
TransitOps is a platform to manage vehicles, drivers, trips, maintenance, fuel expenses, and reports.

## Tech Stack

Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL
