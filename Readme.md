# TransitOps: Smart Transport Operations Platform

TransitOps is a comprehensive full-stack transport management solution designed to streamline fleet operations, driver management, trip dispatching, maintenance tracking, and fuel expense management with role-based access control.

## 🚀 Key Features

*   **Fleet Management**: Real-time monitoring of vehicles with status toggling (Available, In Shop, Retired).
*   **Trip Lifecycle**: End-to-end trip management featuring:
    *   Visual progress tracking (Draft → Dispatched → Completed → Cancelled).
    *   Intelligent **Capacity Validation** (Prevents dispatch if cargo weight exceeds vehicle capacity).
    *   Live Board for real-time dispatch monitoring.
*   **Driver Management**: Manage driver assignments with dynamic status tracking (Available, On Trip, Off Duty, Suspended).
*   **Maintenance & Fuel Tracking**: 
    *   Automated maintenance logs.
    *   Status toggling (In Shop/Completed) which automatically updates vehicle availability status.
    *   Fuel expense logging per trip/vehicle.
*   **Role-Based Access Control (RBAC)**: Secure access tailored to specific organizational roles:
    *   Fleet Manager
    *   Dispatcher
    *   Safety Officer
    *   Financial Analyst
*   **Analytics Dashboard**: Visual insights into fuel expenditure trends and YTD maintenance costs.
